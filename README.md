# NeuroGrip - Assistive Technology App

NeuroGrip is a feature-rich Flutter application designed for assistive technology, focusing on IoT integration and accessibility.

## Features

- **Splash Screen**: Branded entry with smooth animations.
- **Authentication**: Secure login and mandatory onboarding for emergency contacts.
- **Dynamic Dashboard**: Live gesture detection feed with connectivity status indicators.
- **TTS Integration**: Text-to-Speech support for detected gestures.
- **Gesture History**: Detailed log of all performed actions.
- **Advanced Settings**: Control over connectivity, personalization, and device synchronization.

## Tech Stack

- **Flutter & Dart**
- **Provider**: State management
- **Google Fonts**: Typography
- **Animate Do**: Animations
- **Intl**: Date/Time formatting

## How to Run

1.  Ensure you have Flutter installed.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to launch the application.

## Project Structure

- `lib/main.dart`: Entry point and routing.
- `lib/screens/`: All UI screens.
- `lib/providers/`: State management logic.
- `lib/models/`: Data models.
